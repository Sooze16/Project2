# Horus: 
## *Keeping an eye on your home*


## Flow Chart

![Flow](./public/assets/images/Flow.PNG)



## Hardware Requirements

- Raspberry Pi 3b+
- Water Sensor with DAC circuit
- Temperature & Humidity sensor
- PIR Motion Sensor
- Flame Sensor
- Bread Board with GPIO breakout 
- Jumper Wires
